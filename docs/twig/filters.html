Filters

Date format - Required extension: php-intl


{{ '2022-11-02' | datef }}

{#
  Print [en]: November 2, 2022
  Print [pt-BR]: 2 de novembro de 2022
#}



Remove accent


{{ 'ÁÉÍóú são àçô' | remove_accent }}

{# Print: AEIou sao aco #}



Format route


{{ 'ÁÉÍóú são àçô' | route }}

{# Print: aeiou-sao-aco #}



Currency formatting


{{ '1.200,00' | currency_formatting }}

{# Print: 1200.00 #}
